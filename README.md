# Introduction
`ascii-conv` is a simple utility for converting images to ascii

# Usage
Use `ascii-conv <path-to-image> <con/full>` to use and output to stdout an ascii copy of your image. `con` will use a convolution while `full` will output the image in its entirety.

# Example
